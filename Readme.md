>스마힐의 장점
>- 빠른시간내에 기부를 할 수 있다
>- 장소에 구애 받지 않는다
>- 누구나 기부를 요청할 수 있다
>- 기부 하는 사람 받는 사람 모두가 즐겁다

>웃음기부 흐름
>- 기부가 필요한 사용자가 기부를 요청합니다.
>- 웃음을 기부하는 사용자는 마음에 드는 기부 게시물에 웃음을 기부합니다.
>- 웃음을 기부하는 사용자는 재밌는 콘텐츠(사진, 영상)을 통해 자연스럽게 웃을을 기부할 수 있고, 셀카모드를 통해 직접 캡쳐 가능합니다.
>- 기부된 웃음 사진은 처음에 요청한 사용자에게 템플릿을 적용하여 보내줍니다.
